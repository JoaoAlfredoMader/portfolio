    Projeto Empréstimo De Ferramentas

    Durante nosso projeto de conclusão de semestre, eu e mais dois colegas nos dedicamos a solucionar o problema
    de um senhor que buscava gerenciar suas ferramentas, inclusive monitorando quem as utilizava em empréstimos.
    Desenvolvemos o projeto em Java, utilizando a interface Java Swing por escolha do professor.
    Nosso foco foi a implementação das funcionalidades do CRUD, permitindo a gestão completa das ferramentas.
    Além disso, integramos o Google Agenda ao software para facilitar o controle das datas de empréstimos das ferramentas

    Esse definitivamente foi o projeto mais desafiador de todos, tendo que passar por todas as etapas, desde a modelagem
    do software até uma simulação de venda delicadamente detalhada para atender às necessidades específicas do usuário.
    A integração com o Google Agenda trouxe um desafio adicional, exigindo uma compreensão aprofundada da API e a
    implementação de funcionalidades que garantissem uma sincronização eficaz entre o software e a plataforma de agenda.
